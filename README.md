## Проект на Django по рассылке сообщений

Данный проект на Django предоставляет функциональность для рассылки сообщений по электронной почте. 
Он использует библиотеку APScheduler для планирования отправки сообщений в определенное время.

### Установка

1. Клонируйте репозиторий проекта на свой локальный компьютер.
2. Установите требуемые зависимости, выполнив команду pip install -r requirements.txt.
3. Создайте базу данных и настройте подключение к базе данных в файле settings.py.
4. Выполните миграции, запустив команду python manage.py migrate.
5. Создайте суперпользователя admin, используя команду python manage.py csu.
6. Установите все файлы БД (blog.json, mailing.json, users.json)

### Использование

1. Запустите сервер разработки Django с помощью команды python manage.py runserver.
2. Откройте веб-браузер и перейдите по адресу http://localhost:8000/admin/, чтобы получить доступ к административной панели.
3. В административной панели измените рассылку или создайте новую, указав необходимые параметры, такие как время отправки, список получателей и содержимое сообщения.
4. Нажмите кнопку "Сохранить", чтобы добавить рассылку.
5. Планировщик APScheduler запускается командой python manage.py run_mailing.

### Дополнительные функции

- В административной панели вы можете просматривать и редактировать существующие рассылки, пользователей, клиентов и сообщения.
- Создание данных сущностей также доступно из интерфейса самого приложения в браузере.
- При успешной отправке сообщения, соответствующая информация будет записана в журнал приложения.

### Отключение и настройка планировщика

- В коде проекта вы можете изменить временные интервалы планировщика, добавлять новые задачи и настраивать расписание для отправки рассылок.
- Если необходимо отключить планировщик, просто остановите его выполнение, нажав комбинацию клавиш Ctrl + C.
